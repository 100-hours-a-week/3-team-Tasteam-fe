# 알림 목록 정리

이 문서는 현재 FE/BE 구현 기준으로 “알림 목록(인앱 알림)”에서 지원하는 알림과 아직 지원하지 않는 알림을 정리합니다.

## 범위

- 알림 목록(인앱 알림) 노출 대상
- 알림 타입(서버 기준)과 FE 표시 매핑
- 아직 서버에서 생성되지 않는 알림

## 지원하는 알림(현재)

### 서버 타입

- `SYSTEM`

### 생성 트리거

- 그룹 가입 완료 알림
  - BE 이벤트: 그룹 가입 이벤트에서 `NotificationType.SYSTEM` 생성

### FE 노출

- 알림 목록 화면에서 SYSTEM 타입을 “그룹 초대”로 매핑하여 노출
  - 현재 매핑: `SYSTEM -> group_invite`

## 준비된 타입(서버 정의는 있으나 아직 실발송 없음)

### 서버 타입

- `CHAT`
- `NOTICE`

### 현재 상태

- 타입은 정의되어 있으나, 알림 생성 트리거가 추가되어 있지 않음
- FE는 매핑만 존재하며 실제 데이터는 아직 없음
  - `CHAT -> group_activity`
  - `NOTICE -> restaurant_recommendation`

## 지원하지 않는 알림(현 시점)

### 설정 UI에만 존재하는 항목

- 야간 알림 허용
  - BE 스키마/설정 모델에 없음

## 채널 기준 현황

- 인앱(웹) 알림: 지원
  - `notification` 테이블 저장 후 알림 목록에서 조회
- 푸시 알림: 미지원
  - FCM 전송 로직/서비스 없음
- 이메일/SMS: 미지원
  - 채널 enum은 존재하나 발송/설정 정책 미구현
