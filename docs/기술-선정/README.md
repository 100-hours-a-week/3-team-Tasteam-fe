# 프론트엔드 기술 선정

## 런타임

- Node.js: LTS (20.x)

## 프레임워크 / 언어

- React: 19.2.0
- TypeScript: 5.9.x

## 빌드 도구

- Vite: 7.2.x

## 린팅

- ESLint: 9.39.x

## 스타일링

- Tailwind CSS: 4.1.x
- Emotion: 11.14.x

## PWA

- Vite PWA: 1.2.x

## 선정 이유

- Node.js LTS는 안정적이고 널리 지원됩니다.
- React + TypeScript 조합은 타입 안정성이 있는 최신 컴포넌트 모델을 제공합니다.
- Vite는 빠른 개발 서버와 최적화된 빌드 파이프라인을 갖추고 있습니다.
- Tailwind CSS + Emotion으로 유틸리티 기반 + 컴포넌트 범위 스타일링을 제공합니다.
- Vite PWA는 정적 자산 캐싱 중심의 간단한 PWA 환경을 구성합니다.
