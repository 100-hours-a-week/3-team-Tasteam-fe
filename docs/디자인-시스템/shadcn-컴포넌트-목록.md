# shadcn/ui 컴포넌트 목록

이 문서는 `ui.shadcn.com` 문서 기준으로, 사용 가능한 컴포넌트를 빠르게 찾기 위한 인덱스입니다.
기본 링크는 Radix 구현 문서로 연결되며(페이지 내에서 Base UI 등으로 전환 가능), 프로젝트에서는 `shared/ui` 래퍼를 통해서만 사용합니다.

## 빠른 찾기

- 문서 목록 페이지: `https://ui.shadcn.com/docs/components`
- 커뮤니티 레지스트리(추가 컴포넌트): `https://ui.shadcn.com/docs/directory`
