# Tasteam Frontend Setup

This document describes how to set up and run the Tasteam frontend locally.

## Prerequisites

- Node.js LTS (latest stable, currently 20.x) and npm

## Install

```bash
npm install
```

## Run (dev)

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview (after build)

```bash
npm run preview
```
